Pipes incluidos en Angular - Usualmente en el Common Module

Los Pipes en Angular son funciones que permiten transformar los datos de una manera fácil y declarativa directamente en las plantillas HTML. Los Pipes se usan cuando necesitas formatear, modificar o transformar datos antes de mostrarlos en la vista.

Los DecimalPipes se utilizan para formatear números decimales en una representación específica. Puedes controlar el número de decimales, el separador decimal y el separador de miles. Aquí tienes un ejemplo de cómo usarlo:

Precio sin formato: {{ precio }}

Precio formateado: {{ precio | number: '1.2-2'}}

Los CurrencyPipes se utilizan para formatear números como monedas. Puedes especificar el código de la moneda, el símbolo de la moneda y el formato de la moneda. Aquí tienes un ejemplo de cómo usarlo:

Ejemplo con USD (Dolar Americano)

Precio en USD: {{ precio | currency:'':'symbol-narrow':'1.2-2' }}

Ejemplo con PEN (Soles Peruanos)

Precio en PEN: {{ precio | currency:'PEN':'code':'1.2-2' }}

Los PercentPipes se utilizan para formatear números como porcentajes. Puedes especificar el número de decimales y el símbolo del porcentaje. Aquí tienes un ejemplo de cómo usarlo:

Porcentaje sin formato: {{ porcentaje }}

Descuento: {{ porcentaje | percent:'1.0-2' }}

Imagina que estas construyendo una pagina de productos. Aquí tienes un ejemplo de cómo usar estos pipes juntos para mostrar el precio de un producto con descuento en diferentes monedas y el porcentaje de descuento aplicado.

Detalles del Producto

Nombre del Producto: {{ name_prod }}

Precio original: {{ precio | currency:'USD':'symbol':'1.0-2' }}

Descuento aplicado: {{ 0.10 | percent:'1.0-2' }}

Precio final: {{ precio * ( 1 - 0.10 ) | currency:'USD':'symbol-narrow':'1.0-2' }}
